# 記録集制作支援ツール集

このリポジトリには、記録集制作を効率化するためのPythonスクリプト群を格納しています。
事務作業の自動化により、制作時間の短縮とミスの軽減を目的としています。

## 収録ツール

### 1. Word to PDF 一括変換ツール (`Word_to_PDF.py`)
- **概要**: 指定したフォルダ内のWordファイル（.doc, .docx）をすべてPDFに変換します。
- **特徴**: 
  - WindowsのWord APIを利用した高精度な変換。
  - GUIによるフォルダ選択で、直感的な操作が可能。
  - デスクトップをデフォルトの作業場所に設定。

### 2. PDF結合ツール (`pdf_merger.py`)
- **概要**: 複数のPDFファイルを1つのファイルに結合します。
- **特徴**: 
  - **自然順ソート**: ファイル名に含まれる数字（1, 2, 10...）を人間が感じる正しい順番で処理。
  - GUIによるファイル選択・保存先指定。
  - スポンサーページなどのページ順序が重要な資料に最適。

## 開発環境
- **OS**: Windows (Word APIを使用するため)
- **言語**: Python 3.14.0
- **主な依存ライブラリ**: 
  - `pypdf`
  - `pywin32`
