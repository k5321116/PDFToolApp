# PDF Merger 

記録集制作を効率化するための、GUIベースのPDF結合ツールです。
スポンサーページなどの複数のPDFファイルを、ファイル名の数字順（自然順）に結合します。

## 特徴
- **自然順ソート**: `1.pdf, 2.pdf, 10.pdf` を正しい順番で結合します。
- **GUI操作**: ターミナル操作不要で、ファイル選択ダイアログから直感的に操作可能です。
- **ポータビリティ**: Python環境がないPCでも動作するよう、`.exe`化をサポートしています。

## 開発環境
- Python 3.14.0
- ライブラリ: `pypdf`

## 使い方（開発者向け）
1. 依存ライブラリのインストール
   ```bash
   pip install -r requirements.txt
2. スクリプトの実行
   ```bash
   python pdf_merger.py
